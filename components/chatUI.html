I'll provide an implementation of Quicksort in Rust. Quicksort is highly efficient for general-purpose sorting.

fn quicksort<T: Ord>(arr: &mut [T]) {
    let len = arr.len();
    if len < 2 {
        return;
    }
    
    let pivot_index = partition(arr);
    
    // Recursively sort elements before and after pivot
    quicksort(&mut arr[0..pivot_index]);
    quicksort(&mut arr[pivot_index + 1..len]);
}

fn partition<T: Ord>(arr: &mut [T]) -> usize {
    let len = arr.len();
    let pivot_index = len / 2;
    arr.swap(pivot_index, len - 1);
    
    let mut i = 0;
    for j in 0..len - 1 {
        if arr[j] <= arr[len - 1] {
            arr.swap(i, j);
            i += 1;
        }
    }
    
    arr.swap(i, len - 1);
    i
}

Time Complexity Analysis:

  • Average Case: O(n log n)
  • Best Case: O(n log n)
  • Worst Case: O(n²) - occurs when the array is already sorted and we pick the last element as pivot.
  • Space Complexity: O(log n) auxiliary stack space.
